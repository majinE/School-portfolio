Project 3: GUN CONTROL: BAN ALL FIREARMS?

Reflection

Project 3 has been the most difficult paper I have had to write so far in my life. The required amount of writing was what I expected from a college course, but the amount of biased sources I had to work with was extremely challenging. I would find sources contradicting each other left and right; conservative blogs attacking liberal blogs and vice-versa on why one another was incorrect. I found myself using my own knowledge to make an estimate on the cost of some gun control law that Democrats have proposed. I discovered that my skills continued to expand; It was becoming easier to pinpoint all the information that was biased, skewed, or incorrect using several resources to validate my predictions. Making the website is by far the best part of the project. Incorporating some type of Computer Science into this project made it even easier to enjoy as it eased the headaches I would get from finding useful sources.
